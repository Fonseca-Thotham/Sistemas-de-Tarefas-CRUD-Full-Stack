# 📝 Sistema de Tarefas (CRUD Full Stack)

Aplicação Full Stack desenvolvida com **React (frontend)** e **Node.js + Express (backend)**, permitindo gerenciar tarefas com operações completas de CRUD.

---

## 🚀 Funcionalidades

* ✅ Criar tarefas
* 📋 Listar tarefas
* ✏️ Editar tarefas
* ❌ Excluir tarefas

---

## 🛠️ Tecnologias utilizadas

### Frontend

* React
* Axios
* CSS

### Backend

* Node.js
* Express
* CORS

---

## ⚙️ Como executar o projeto

### 🔹 Backend

```bash
cd backend
npm install
node server.js
```

Servidor rodando em:

```
http://localhost:3001
```

---

### 🔹 Frontend

```bash
cd frontend
npm install
npm start
```

Aplicação rodando em:

```
http://localhost:3000
```

---

## 💡 Sobre o projeto

Este projeto foi desenvolvido com o objetivo de praticar conceitos de desenvolvimento Full Stack, incluindo:

* Integração entre frontend e backend
* Consumo de API REST
* Gerenciamento de estado no React
* Organização de código e versionamento com Git
